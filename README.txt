Raiti Project

----------
2018년(고등학교)에 공부한 C 프로그래밍을 토대로 제작한 슈팅게임 프로젝트
플래쉬 게임이었던 Raiden을 모티브로 하여 슈팅 게임을 개발하고자 하였음.

이 프로젝트는 C 프로그래밍 수업 과제 제출용으로 제작했다. 매우 짧은 기간동안 제작하여야 했기에 미흡한 점이 매우 많다.
직접 코드를 뜯어 간단하게 자신의 비행기를 모델링하거나, 코드를 추가하여 특정 레벨에 나올 적 비행기(체력, 이동방향, 움직임 속도, 총알 속도 등)를 구성할 수 있다.

main.c 코드를 뜯어보면 나름 코드 수정 및 추가에 있어 편의성을 고려하여 코드를 구성하였다.

안타까운 점은, 배경음과 총알을 발사할 때의 효과음 코드가 알 수 없는 오류로 인해 동작하지 않는다.(에러) 때문에 임시방편으로 관련 명령어를 모두 주석처리 해 놓았다. Git에서는 사운드파일을 모두 삭제해 놓았다.
----------


--플레이 방법--
본격적으로 게임을 시작하면 레벨과 제한시간이 있는데, 레벨마다 생성되는 적 비행기가 각각 존재한다.
특정 레벨에서 다음 레벨로 넘어가려면 제한시간 내 남아있는 적 비행기를 모두 없애야 한다.
적의 총알, 나의 총알 모두 기본적으로 '*'이며, 코드 수정또한 가능하다.
적 비행기는 체력이 있으며, 체력을 모두 깎아야만 쓰러뜨릴 수 있다.
반면, 플레이어 비행기는 적 총알을 한 발만 맞아도 죽는다.


--조작법--
화살표 : 자신의 비행기 조종
Z, X : 총알 발사
